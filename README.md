# 스팀톡톡(Steem Talk Talk)
스팀에서 토크하자. 

## 개발환경
```
Node.js v10.15.3
npm 6.4.1
yarn 1.16.0
expo-cli 3.0.10
```

## 설치하기

```bash
$ git clone https://github.com/SteemDeveloper/steemtalktalk.git
$ cd steemtalktalk
$ npm install expo-cli yarn --global
$ yarn
```

## 실행하기

### 아이폰
```bash
$ yarn ios
```

### 안드로이드
```bash
$ yarn ios
```

### 웹
```bash
$ yarn web
```
